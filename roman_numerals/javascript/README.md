## Requirements:

* Ruby 1.9.2+
* [Phantomjs](http://phantomjs.org). Install phantomjs from <http://phantomjs.org>.
  If you use MAC OS X and have brew installed, you can use `brew install phantomjs`. 

## Running tests from the terminal:

Execute:

    rake test
